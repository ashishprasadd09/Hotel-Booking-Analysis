# Hotel Booking Analysis
#### Exploratory Data Analysis and Data Visualization in Python
#
## Description: 

 Aim of this project is to explore and analyse reason for canceled room bookings. To find correlation between a variable and how it leads to booking cancellation. Data was visualized to better understand findings.
